IntelliJ IDEA 유용한 15가지 단축키 알아보기

안녕하세요 이번 시간에는 유튜브 IntelliJ IDEA by JetBrains 공식 채널에서 제공한 인텔리제이 아이디의 15가지 도움이 되는 단축키들을
알아보도록 하겠습니다. 인텔리제이는 우리가 앞으로 개발자로 살아가는데에 있어서 좋은 개발툴입니다. 이번 시간에 유용한 단축키들을 배우면
앞으로 편안할 일들이 많아지겠죠? 해당 내용은 아래의 표로 자세히 살펴보겠습니다. 맥북은 사용해보지 않아 윈도우 기준으로 설명드리겠습니다.

No. 단축키 기능설명 사용예시
1 Alt+Enter 하이라이트된 부분에 대한 각종 suggestion을 제공.
  • 오류가 있는 부분에 Alt+Enter를 통해 오류 해결책 제시
  • static import를 자동 생성
2 F2 다음 에러포인트로 이동시켜줌 코드관리 시에 어떤 쪽에서 에러가 나는지 빠르게 하나씩 넘어가면서 체크 가능
3 Alt+1 프로젝트 메뉴를 온오프 전체화면으로 코딩을 하고 싶을때 프로젝트 메뉴를 접어두거나 켜거나 할 수 있음
4 ESC 모든 팝업 상태 또는 코드 라인에서 벗어난 상태에서 누르면 다시 커서를 코드 라인으로 돌아가게 한다. 팝업메뉴 좀 보다가 빠르게 끄고 코드로 돌아갈 때 사용
5 Ctrl+E 최근파일, 인텔리제이 하단의 툴들을 빠르게 오픈할 수 있는 단축키 최근 파일 빨리 열어볼 때 사용
6 Crtl+B 특정 변수 또는 함수,클래스들의 선언부로 빠르게 이동시켜주는 단축키 특정 함수 또는 변수,클래스가 어떤 형태로 짜여져 있는 지 확인할 때 사용
7 Alt+F7 하이라이트된 부분이 전체 프로젝트 파일중에 어느 부분에서 쓰이고 있는지 총망라 해당 변수 또는 클래스,함수가 어떤 곳에 쓰이는 지 확인하고 수정할 때 사용할 수 있음
8 Ctrl twice 특정 파일을 입력하여 실행시킬 수 있는 단축키 작성중인 파일이나 다른 테스트 파일들을 빠르게 실행시켜볼 때 사용할 수 있음
9 Ctrl+W 하이라이트된 부분에서 단어부터 태그, 상위 영역등으로 스케일을 업시켜주면서 하이라이트로 감싸올라가는 단축키 특정부분을 빠르게 지우거나 복사할 때 활용할 수 있음
10 Ctrl+/ 하이라이트된 열을 전부 주석처리 시킬 수 있는 단축키 주석처리가 필요한 부분을 드래그하여 빠르게 주석처리할 수 있다.
11 Shift+Ctrl+Enter 작성중인 여러 문의 틀을 자동완성해주는 단축키 for문이나 if문 사용시 틀을 빠르게 완성시켜 코드를 작성할 수 있음
12 Ctrl+Alt+L 코드의 형태별로 정제하여 코드를 정리하는 단축키 코드가 복잡할 때 좀 더 클린하고 명확하게 작성하기 위해 사용
13 Shift+Ctrl+Alt+T 사용가능한 모든 리팩토링을 보여주는 단축키 코드를 리팩토링할 때 참고하여 사용
14 Shift+Ctrl+A 인텔리제이의 액션뿐만이 아닌 단축키들을 볼 수 있는 단축키 특정 액션의 내용을 검색하거나 단축키를 찾을 때 사용할 수 있음
15 Shift twice 파일, 액션, 단축키 포함 모든 내용들을 검색할 수 있는 단축키 찾아보려는 내용이 어느 곳에 속한지 모를 때 모두 검색 기능을 사용할 수 있음

이상으로 인텔리제이의 유용한 단축키들을 알아보았습니다. 열람해주셔서 감사합니다!
